Here are some of my projects

  • Viber

    JavaScript, React, Node.js, CSS, Git

    Viber is a full stack web application that allows users to add a playlist populated with songs of their choice to their Spotify account. A Spotify account is needed in order to use this application. Once logged in, search up any artist or song and add it to the new playlist. Change the name of the playlist if desired and press the "Save to Spotify" button to have a new playlist ready for listening!

  • Munchie

    JavaScript, React, Node.js, CSS, Git

    Munchie is a full stack web application that allows users to search up any type of food business with an included location to see all restaurants in that given location. Users can sort how the restaurants are listed as well. Restaurants have the option of being sorted by best match, highest rated, and most reviewed. If a restaurant happens to catch your eye, clicking on the picture will also lead you to the restaurant's official Yelp page!

  • Texas State Desktop App

    Java, JavaFX, MySQL, Jaunt, TravisCI, Git

    This project is a semester long project for my software engineering class that allows users (Texas State University students) to view university services such as buses, food locations, academic work, and so on. Project was done in collaboration with four other students utilizing continuous deployment, storyboard, sprints, and Git.
